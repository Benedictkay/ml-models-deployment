# Spotify Playlist Generation (KMeans)

A compact Streamlit app that generates playlist recommendations using K‑means clustering on Spotify track features. Use the sliders in the UI to input song features and get cluster-based recommendations.

Run locally:

```bash
pip install -r requirements.txt
streamlit run app.py
```

That's it — the live app is linked at the top.
